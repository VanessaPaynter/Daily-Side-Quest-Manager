## 🌌 What Is This?

Daily Side Quest Manager turns your boring to-do list into a tiny pixel RPG.
Designed for tasks that take 5 minutes or less. Why? Because small wins add up, and starting should feel easy, fun, and pressure-free.
If you can do it in under 5 minutes, might as well get it out of the way!

Finish a task → gain XP → YEET a rocket into the stratosphere with spark bursts and confetti explosions.

## 💻 Run It (Dev Mode)

```bash
git clone https://github.com/VanessaPaynter/Daily-Side-Quest-Manager.git
cd Daily-Side-Quest-Manager
npm install
npm run dev
```

---

## 🖥 Run As Desktop App

```bash
npm run build
npm run electron
```

---

## 📦 Build a Shareable Desktop App

```bash
npm run build
npx electron-packager . SideQuestBuddy --platform=darwin --arch=arm64 --out=release-build --overwrite
```
